Tag Stream Settings

  Tag stream lets you publish a copy of your Data Catalog tags to Pub/Sub. As you create new tags through Tag Engine, a copy of those tags is written into a Pub/Sub topic.
  Tags for all tag templates are written into the same topic.

  Enter the Pub/Sub details for your tag stream settings. Tag Engine will not create the Pub/Sub topic or subscription, you must create those outside of Tag Engine before proceeding.
