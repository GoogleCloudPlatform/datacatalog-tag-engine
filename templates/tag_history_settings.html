Tag Engine for Data Catalog

Tag History Settings

  Tag history lets you save a copy of your Data Catalog tags in BigQuery. As you create new tags through Tag Engine, a copy of those tags is written into one or more BigQuery tables.
  Tag Engine creates one BigQuery table per tag template and writes one record in this table for each tag update.

  Enter the BQ details for your tag history tables. Tag Engine will not create the BQ dataset, you must create it outside of Tag Engine before enabling tag history.
