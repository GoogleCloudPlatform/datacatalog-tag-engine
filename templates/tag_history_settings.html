Tag History Settings

  Tag History lets you save a copy of your Data Catalog tags in BigQuery. As you create or update tags through Tag Engine, a copy of those tags is written into one or more BigQuery tables.
  Tag Engine creates one BigQuery table per tag template and writes one record in this table for each tag create or update.
